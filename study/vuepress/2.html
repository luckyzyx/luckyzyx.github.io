# 正式开始

# 全局安装VuePress

yarn global add vuepress # 或者：npm install -g vuepress
1

新建一个文件夹作为工作目录

这里以E:\work\为例

Git安装完成后右键菜单会出现Git bash here

在Work文件夹内右键选择bash

# 初始化项目

yarn init -y # 或者 npm init -y
1

# 创建docs文件夹

在work文件夹下新建docs文件夹作为项目文档根目录，主要放置Markdown类型的文章和.vuepress文件夹。

# 创建READMD.md文件

输入以下代码

---
home: true
heroImage: /hero.png
actionText: 快速上手 →
actionLink: /zh/guide/
features:
- title: 简洁至上
  details: 以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。
- title: Vue驱动
  details: 享受 Vue + webpack 的开发体验，在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。
- title: 高性能
  details: VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。
footer: MIT Licensed | Copyright © 2018-present Evan You
---

# 创建.vuepress目录

在docs目录中，创建.vuepress目录，.vuepress目录主要用于存放VuePress相关的文件。

# 创建config.js

进入到.vuepress目录中，然后创建config.js文件

config.js是VuePress必要的配置文件，它导出y一个javascript对象。

# 创建public文件夹

进入到.vuepress目录中，然后创建public文件夹，此文件夹主要放静态资源文件，例如favicons和PWA的图标。

# 项目结构

work
├─── docs
│   ├── README.md
│   └── .vuepress
│       ├── public
│       └── config.js
└── package.json
