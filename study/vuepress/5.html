# Vuepress 插件

# 图片缩放

medium-zoom插件

# 安装

yarn add -D @vuepress/plugin-medium-zoom
# OR npm install -D @vuepress/plugin-medium-zoom
1
2

# 使用

# 简单使用

module.exports = {
  plugins: ['@vuepress/medium-zoom']
}
1
2
3

# 自定义选项:

module.exports = {
  plugins: {
    '@vuepress/medium-zoom': {
      selector: 'img.custom',
      // medium-zoom options here
      // https://github.com/francoischalifour/medium-zoom#options
      options: {
            margin: 0,//放大图像外部的空间
            background: '#fff',//覆盖图的背景
            scrollOffset: 40, //滚动以关闭缩放的像素数
            container: null,//呈现放大的视窗
            template: null,//缩放时显示的模板元素
        }
    }
  }
}

# 选项

# selector

  • 类型: string
  • 默认值: .theme-default-content :not(a) > img

值得注意的是， .theme-default-content 是默认主题添加给 <Content /> 组件的 class name

# 实例展示

<img class="custom" src="/assets/logo.jpg" >
